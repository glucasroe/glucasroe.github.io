# G. Lucas Roe's Portfolio & Digital Garden

Portfolio splash page, blog, and digital garden for G. Lucas Roe, a product design leader from Maryland.

Built with [Astro](https://astro.build) and hosted on GitHub Pages.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- Static site generation with Astro
- Content collections for blog posts and digital garden notes
- Wikilink support for interconnected notes
- Obsidian vault integration
- Custom domain: glucasroe.com
